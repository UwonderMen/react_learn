#购物车实现流程
```
    点击购物车后，向服务器发送请求，让服务器存储当前新加的信息（为了防止页面刷新后，购物车的信息消失了，即redux中的信息在页面刷新后没有了）

    当服务器返回存储成功后，我们把信息往redux中存储一份（建议最好是从服务器获取我的购物车信息，存储到redux中：为什么？
        1、存储到redux中，目的是为了以后切换到我的购物车页面的时候，没必要再去服务器端拉取数据，直接从redux获取数据性能更高
        2、之所以是从服务器获取信息存储到redux中，因为服务器的信息是最准确的，即使页面刷新了，即使某些操作忘记向redux中存储了，每一次都可以获取到最新信息（只要你向服务器请求数据））
```
